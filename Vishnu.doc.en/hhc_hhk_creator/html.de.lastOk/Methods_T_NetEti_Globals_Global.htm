Methoden
 NameBeschreibung
CompareVersion

Vergleicht zwei Versionsstrings, die aus durch '.' separierten Zahlen bestehen, z.B.: 1.0.0.10 mit 1.0.0.9.

Compares two version-strings, consisting of '.' separated lines, i.e.: 1.0.0.10 mit 1.0.0.9.

DirectoryCopy Kopiert ein Verzeichnis inklusive aller darin enthaltenen Dateien. Wenn der Parameter "copySubDirs" auf true steht, werden auch alle im Verzeichnis enthaltenen Unterverzeichnisse rekursiv mit kopiert.
GetUniversalName

Konvertiert einen übergebenen absoluten Dateipfad in sein UNC-Äquivalent, wenn möglich.

Converts a given absolute file path into it's UNC-equivalent, if possible.

IsDate

Returnt true, wenn der übergebene String ein gültiges Datum ist.

Returns true, if the given string is a valid date.

StringToSecureString

Wandelt einen übergebenen String in einen SecureString

Converts a given string to a SecureString

UnicodeHexcodeToChar

Wandelt eine Zeichenkette, die einen Unicode-Hexcode darstellt z.B.: 20AC (=> €) in das entsprechende Zeichen um.

Converts a string, representing a hex-unicode character like 20AC, to ascii €.
