PropertiesChangedEventArgs Klasse

EventArgs für das Event "PropertiesChanged" aus "INotifyPropertiesChanged". Das Event "PropertiesChanged" kann von LogicalNodeViewmodel abonniert werden. Dieses erhält über die übergebenen PropertiesChangedEventArgs eine String-List mit Property-Namen und kann seinerseits über INotifyProperyChanged die UI informieren.
