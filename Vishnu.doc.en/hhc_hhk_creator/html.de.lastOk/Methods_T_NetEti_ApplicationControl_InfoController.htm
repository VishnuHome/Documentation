InfoController Methoden

Der InfoController Typ exponiert die folgenden Member.

Methoden
 NameBeschreibung
Dispose Öffentliche Methode zum Aufräumen.
Dispose(Boolean) Hier kann ggf. aufgeräumt werden.
DisposeAll Disposed alle Viewer.
Finalize Finalizer: wird vom GarbageCollector aufgerufen.
(Überschreibt ObjectFinalize)
Flush Flusht alle flushable Viewer/Logger.
FlushAll Hierüber werden alle InfoPublisher geflusht.
GetInfoController Returnt die statische Property der nestet Klasse NestedInstance (als IInfoController), welche ihrerseits den privaten Konstruktor aufruft, wenn noch keine Instanz existiert.
GetInfoPublisher Returnt die statische Property der nestet Klasse NestedInstance (als IInfoPublisher), welche ihrerseits den privaten Konstruktor aufruft, wenn noch keine Instanz existiert.
GetInfoSource Returnt die statische Property der nestet Klasse NestedInstance (als IInfoSource), welche ihrerseits den privaten Konstruktor aufruft, wenn noch keine Instanz existiert.
Publish(Object) Hierüber wird eine neue Message des Typs InfoType.Info mit Absender null verbreitet.
Publish(Object, Object) Hierüber wird eine neue Message des Typs InfoType.Info verbreitet.
Publish(Object, Object, InfoType) Hierüber wird einen neue Message verbreitet.
RegisterInfoReceiver(IInfoViewer, InfoType) Hierüber trägt sich ein interessierter Viewer in die Liste der zu informierenden Viewer ein.
RegisterInfoReceiver(IInfoViewer, Type, InfoType) Hierüber trägt sich ein interessierter Viewer in die Liste der zu informierenden Viewer ein.
Say Hierüber wird 'mal eben' einen Message verbreitet.
Show Hierüber werden alle Logs ausgegeben.
UnregisterInfoReceiver Hierüber meldet sich ein eingetragener Viewer wieder ab.
