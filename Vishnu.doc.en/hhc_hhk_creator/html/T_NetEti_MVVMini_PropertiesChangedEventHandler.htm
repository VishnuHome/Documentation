PropertiesChangedEventHandler Delegate

"PropertiesChangedEventHandler" defines the type of an event handler routine for the "PropertiesChanged" event. "PropertiesChanged" can be used by LogicalNodeViewmodel can be subscribed to. This is received via the transferred "PropertiesChangedEventArgs" a string list with property names and can in turn be changed via INotifyProperyChanged inform the UI.
