Version 7.0.1.6

Version 7.0.1.6 Hard Core (as of October 16, 2023)

This version is the first Vishnu software to be completely migrated to .Net 7.0.
This also includes the framework, all plugins and the documentation. This version is tested and stable.

Changes in this version

  • The Vishnu-Installer (ClickOnce) was deactivated after migration to .Net 7. A better solution must be found here in the long term.

  • The VSIX-Templates to support the creation of your own Vishnu-Checkers and User controls have been made functional again after migration to .Net 7, but show error messages when generating new projects. Nevertheless, everything works.

  • Creation of NuGet packages via a universal solution for all projects controlled by an environment variable NETETI_CIB=true and ...\AddOn\proj\Conditional_NuGet_Package_Properties.proj is implemented.

  • MicroMailer (and all other workers): Parameters revised and, where necessary, outsourced.

  • Vishnu positioning and resizing with multiscreen orientation (-SizeOnVirtualScreen=true) revised.

  • The orientation of the Vishnu-tree can now be changed at runtime using F4 in addition to clicking the YinYang symbol.

  • Vishnu-specific messages with correct positioning, zoom functions and icons implemented.

  • ToolTips with copy function realised.

  • InPlug/WebScraper: own solution via shortening and counting down the Chrome version implemented to download a suitable driver version.
