Version 5.0.1.6

Version 5.0.1.6 HotPlug (as of February 09, 2021)

Changes in this version

  • Writing and loading snapshots optimized, secured and retested.

  • Ring logger implemented - solution via parameterizable renaming and deletion of excess log files.

  • The maximum opening time for tooltips has been increased to 30 seconds.Environment parameters provided with Vishnu_ prefix.

  • Writing and loading of snapshots optimized, secured and retested. Ring Logger implemented - solution via parameterizable renaming and deletion of surplus log files. The maximum opening time for ToolTips has been increased to 30 seconds. Environment parameter with Vishnu_ prefix.

  • --- This functionality is still to be regarded as experimental, comprehensive tests are still pending ---
    Dynamic snapshots and trees - TreeReload implemented. Context menu on right mouse button (for this, "Normalize view" moved to Shift-right mouse button):

    • Reload tree

    • Stop the tree and let it continue running

    • Log tree or branch

  • UserControls with ListViews with unhandled SelectionChanged could lead to Vishnu being aborted in the MainTabControl because SelectionChanged bubbled up in the tree.

  • Bugfix: Direct UserRun on triggered nodes led to double start of the nodes, because the assigned trigger was also triggered again immediately.
