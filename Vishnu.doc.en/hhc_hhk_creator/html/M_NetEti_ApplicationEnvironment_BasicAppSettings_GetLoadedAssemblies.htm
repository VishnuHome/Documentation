BasicAppSettingsGetLoadedAssemblies method

Returns a dictionary that is used for all assemblies loaded in the application. contains the FullName and the assembly. Can be helpful for troubleshooting in certain cases.
Note: If the switch "DumpLoadedAssemblies" = true, the keys of this dictionary (FullName) are logged in "Dispose".
