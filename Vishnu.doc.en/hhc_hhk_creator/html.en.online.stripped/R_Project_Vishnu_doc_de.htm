Vishnu - technical Documentation
This part of the Vishnu documentation is aimed at software developers or users who have basic knowledge of .Net software development. The application-oriented part of the documentation can be found at Vishnu - Observe and control processes.
Namespaces
Namespace Description of the
LogicalTaskTree Main processing logic of Vishnu
LogicalTaskTree.Provider Provider for Vishnu jobs
NetEti.ApplicationControl General routines for application control
NetEti.ApplicationEnvironment Determination of environmental parameters of the application
NetEti.CustomControls Own or customised controls for use in Vishnu
NetEti.ExpressionParser Processing routines for logical expressions
NetEti.FileTools Routines for access via the file system
NetEti.FileTools.Zip ZIP Pack/Unpack routines
NetEti.Globals Global types, constants, classes and static functions
NetEti.MultiScreen Support for the use of multiple screens
NetEti.MVVMini Basic functionalities for MVVM applications
