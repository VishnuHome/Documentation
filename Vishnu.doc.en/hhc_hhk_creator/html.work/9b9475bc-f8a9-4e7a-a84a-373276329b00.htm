Vishnu.exe.config.user

Die Datei Vishnu.exe.config.user ist die zentrale Stelle für Ihre eigenen Vishnu-Einstellungen. Hier finden Sie einige voreingestellte Parameter. Sie können diese ändern, löschen oder auch eigene hinzufügen. Sie finden die Vishnu.exe.config.user per Grundeinstellung im Pfad %LOCALAPPDATA%\Vishnu\Vishnu.exe.config.user, wobei LOCALAPPDATA eine Einstellung der Windows-Umgebung (Environment) ist und üblicherweise auf C:\Users\<Benutzer>\AppData\Local verweist.

Beispiel für eine Einstellung:
XML: <add key="Job" value="DemoJobs\Simple\CheckAll" />
Json: "Job": "DemoJobs\Simple\CheckAll",

  Hinweis

Auf der Seite Konfiguration und Parameter finden Sie weitere Möglichkeiten, Vishnu-Parameter dauerhaft oder vorübergehend zu setzen.

Vishnu.exe.config.user nach Installation

So sieht die Konfigurationsdatei direkt nach dem ersten Start von Vishnu aus:

XML
Json
Copy
<?xml version="1.0"?>
<configuration>
  <appSettings>
    <!--   Benutzeranwendung und konfigurierte Eigenschafteneinstellungen hier einfuegen.-->
    <!--   Beispiel: <add key="settingName" value="settingValue"/> -->
    <add key="UserParameterReaderPath" value="%UserAssemblyDirectory%\DemoParameterProvider.dll|PARA-1|PARA-2|M:1" />
    <add key="DumpAppSettings" value="true"/>
    <add key="Autostart" value="true"/>
  </appSettings>
</configuration>

{
  "UserParameterReaderPath": "%UserAssemblyDirectory%\\DemoParameterProvider.dll|PARA-1|PARA-2|M:1",
  "DumpAppSettings": true,
  "Autostart": true
}
